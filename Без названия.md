- Осторожно при беременности (может стимулировать матку)

**Вариант 2: Скрипт для автоматического создания файлов**

Если вы хотите автоматизировать процесс, вот скрипт на Python, который создаст папку `здоровье` и три файла внутри нее:

```python
import os

# Создаем папку, если ее нет
os.makedirs("здоровье", exist_ok=True)

files = {

- Ветрогонное

## Применение
- Наружно: примочки, компрессы, полоскания
- Внутрь: чай при расстройствах ЖКТ
- Косметология: ополаскивание волос

## Противопоказания
- Индивидуальная непереносимость
- Осторожно при беременности
""",

    "здоровье/мята.md": """# Мята перечная (Mentha piperita)

## Описание
Многолетнее травянистое растение семейства Яснотковые.

- Кулинария

## Противопоказания
- Гипотония
- Варикозное расширение вен
- Детям до 3 лет
""",

    "здоровье/крапива.md": """# Крапива двудомная (Urtica dioica)

## Описание
Многолетнее травянистое растение семейства Крапивные.

## Полезные свойства
- Кровоостанавливающее
- Мочегонное
- Общеукрепляющее
- Антиаллергическое

## Применение
- Отвар крапивы
- Наружно: ополаскивание волос
- Кулинария

## Противопоказания
- Повышенная свертываемость крови
- Гипертония
- Беременность
"""
}

# Записываем файлы
for filename, content in files.items():
    with open(filename, "w", encoding="utf-8") as f:
        f.write(content)
    print(f"Создан файл: {filename}")

print("Готово! Файлы созданы в папке 'здоровье'")
```

**Как использовать скрипт:**
1. Сохраните код в файл (например, `create_files.py`)
2. Запустите его: `python create_files.py`
3. В папке, где находится скрипт, появится папка `здоровье` с тремя файлами `.md`
